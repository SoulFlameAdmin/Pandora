PANDORA LOCAL FOLDER

Файлове:
- index.html -> локален вариант с отделни style.css и script.js
- source_original.html -> оригиналният .htm от ZIP-а
- style.css -> всички inline CSS блокове, извадени отделно
- script.js -> всички inline JS блокове, извадени отделно и wrapped в try/catch

Как да го ползваш:
1. Копирай цялата папка `pandora` в E:\pandora
2. Отвори E:\pandora\index.html

Важно:
Оригиналният ZIP не съдържа папката `pandora -sf_files`, затова някои външни Slider Revolution ресурси не бяха вътре.
Добавен е bootstrap loader, който пробва да зареди публични библиотеки автоматично.
Затова това е максимално близък local extraction вариант от качения ZIP.
